## 1. 시스템 구성

### 페이지 안내

| 페이지 | 주소 | 용도 |
|--------|------|------|
| Admin (대회 관리) | `/tournament/manage-3v3/{대회ID}` | 대회 생성, 경기 등록, 기록, 저장 |
| Admin (직접 접속) | `/threevthree/admin` | 위와 동일 (대회 선택 가능) |
| 기록 열람 | `/records` | 대회 결과 공개 조회 (HALL OF FAME) |
| 자유 전광판 | `/threevthree` | 누구나 쓸 수 있는 전광판 (DB 저장 안 됨) |

### 핵심 개념

- **자유 전광판** (`/threevthree`): 연습, 픽업게임용. 점수판만 제공하며 결과가 저장되지 않음.
- **Admin 대회 관리**: 공식 대회 전용. 경기 등록 → 기록 → 저장 → DB 반영. HALL OF FAME에 노출됨.

## 2. 대회 전 준비

### 2-1. Admin 페이지 접속

Tournament Dashboard (`/tournament/dashboard`)에서 해당 3v3 대회의 **관리** 버튼을 눌러 Admin에 진입합니다.

또는 직접 `/threevthree/admin` 주소로 접속할 수 있습니다.

### 2-2. 대회 생성

1. **"대회 관리"** 섹션에서 대회명을 입력합니다.
   - 예: `2026.03 GritLab 3:3`, `4월 정기 3v3`
2. **"생성"** 버튼을 클릭합니다.
3. 생성 즉시 해당 대회가 선택되고 "진행중" 상태가 됩니다.

> 이미 대회가 있다면 드롭다운에서 선택하면 됩니다.

### 2-3. 경기(대진) 등록

1. **라운드 탭**에서 해당 라운드를 선택합니다.

   | 탭 | 용도 |
   |----|------|
   | 예선 A | 예선 A조 경기 |
   | 예선 B | 예선 B조 경기 |
   | 8강 | 8강 토너먼트 |
   | 4강 | 4강 토너먼트 |
   | 3위전 | 3/4위 결정전 |
   | 결승 | 결승전 |

2. **"경기 추가"** 버튼을 클릭합니다.
3. 경기 카드가 생성되면 **팀 A / 팀 B 이름**을 입력합니다.
4. 필요한 만큼 경기를 추가합니다.

### 2-4. 대진표 확인

경기를 등록하면 페이지 하단에 **토너먼트 대진표**가 자동으로 생성됩니다.
라운드별로 시각화되어 참가팀과 결과를 한눈에 확인할 수 있습니다.

## 3. 대회 당일 운영

### 운영 흐름

```
경기 등록 (팀명 입력)
    ↓
"기록 시작" 버튼 클릭
    ↓
라이브 스코어보드에서 경기 진행
    ↓
경기 종료 → "저장 & 종료" 클릭
    ↓
결과가 DB에 즉시 저장
    ↓
다음 경기로 이동
    ↓
모든 경기 완료 → "대회 종료" 클릭
```

## 4. 경기 기록 (라이브 스코어보드)

### 4-1. 기록 시작

경기 카드에서 팀명이 입력된 상태에서 **"기록 시작"** (초록색 버튼)을 클릭하면 전체 화면 스코어보드로 전환됩니다.

### 4-2. 스코어보드 조작법

#### 게임 클락 (GAME CLOCK)

| 조작 | 동작 |
|------|------|
| **탭 (터치)** | 타이머 재생 / 일시정지 |
| **롱프레스 (0.4초 이상)** | 타이머 정지 → 시간 설정 모달 표시 |

- 시간 설정 모달에서 **분 단위**로 입력 후 "확인"을 누르면 게임클락이 변경됩니다.
- 기본값: **10분** (600초)

#### 샷 클락 (SHOT CLOCK)

| 조작 | 동작 |
|------|------|
| **탭 (터치)** | **12초로 리셋** |

- 기본값: **12초** (FIBA 3x3 규정)
- 샷클락 5초 이하 → 빨간색으로 변경
- 샷클락 0초 도달 → 부저 자동 재생

#### 점수

| 조작 | 동작 |
|------|------|
| **점수 숫자 터치** | +1점 |
| **+ 버튼** | +1점 |
| **- 버튼** | -1점 (최소 0) |

#### 팀 파울

| 조작 | 동작 |
|------|------|
| **파울 숫자 터치** | +1 파울 |

- 7 파울 이상 → **PENALTY** 표시 (빨간색 강조)

#### 부저

- 헤더의 **종(벨) 아이콘**을 눌러 수동 부저를 울릴 수 있습니다.
- 게임클락 0초, 샷클락 0초 도달 시 자동으로 부저가 울립니다.

### 4-3. 경기 저장

경기가 끝나면 헤더 오른쪽의 **저장 아이콘 (디스크 모양)**을 클릭합니다.

- 현재 점수가 DB에 즉시 저장됩니다.
- 승자가 자동으로 판별됩니다 (A 승 / B 승).
- 저장 후 Admin 관리 화면으로 돌아갑니다.

> 저장하지 않고 나가려면 헤더 왼쪽의 **← (뒤로가기)**를 누릅니다. 이 경우 점수가 저장되지 않습니다.

### 4-4. 점수 직접 입력 (수동)

라이브 스코어보드를 사용하지 않고, 경기 카드에서 직접 점수를 입력할 수도 있습니다.

1. 경기 카드의 점수 입력란에 최종 점수를 직접 입력합니다.
2. **"저장"** 버튼을 클릭합니다.
3. 승자가 자동 판별되어 DB에 저장됩니다.

## 5. 대회 종료 처리

### 5-1. 대회 종료

모든 경기가 완료되면, 헤더 오른쪽의 **"대회 종료"** 버튼 (빨간색)을 클릭합니다.

- 대회 상태가 "진행중" → "종료됨"으로 변경됩니다.
- HALL OF FAME에서 "(종료됨)"으로 표시됩니다.

### 5-2. 대회 재개

실수로 종료했거나 추가 경기가 필요한 경우:

1. Admin 페이지에서 해당 대회를 선택합니다.
2. 헤더의 **"대회 재개"** 버튼 (초록색)을 클릭합니다.
3. 대회 상태가 다시 "진행중"으로 변경됩니다.

### 5-3. 경기 삭제

잘못 등록한 경기는 경기 카드의 **"삭제"** 버튼 (빨간색)으로 삭제할 수 있습니다.
확인 팝업이 나타나며, "확인"을 누르면 영구 삭제됩니다.

## 6. 기록 확인 (HALL OF FAME)

### 접속 방법

- 메인 페이지에서 **"RECORDS"** 메뉴 클릭
- 또는 직접 `/records` 주소 접속

### 대회 선택

상단 **"Select Season"** 드롭다운에서 대회를 선택합니다.

- `[3v3]` 접두사: 3:3 대회
- `[3PT]` 접두사: 3점슛 대회
- `(진행중)`: 현재 진행 중인 대회
- `(종료됨)`: 완료된 대회

### 3v3 대회 조회 시

- **대진표 뷰**: 라운드별 대진 결과를 시각적으로 표시
- **경기 리스트**: 전체 경기의 라운드, 대진, 스코어, 승자를 테이블로 표시
- 승리 팀은 주황색 하이라이트로 강조

## 7. 자주 묻는 질문 (FAQ)

### Q. 자유 전광판과 대회 관리의 차이는?

| 항목 | 자유 전광판 (`/threevthree`) | Admin 대회 관리 |
|------|---------------------------|----------------|
| 접근 | 누구나 | 운영자 |
| 데이터 저장 | X (로컬만) | O (DB 저장) |
| HALL OF FAME | X | O |
| 대진표 | X | O |

### Q. 경기 도중 시간을 변경하고 싶어요.

게임클락을 **0.4초 이상 길게 누르면(롱프레스)** 시간 설정 모달이 나타납니다.
원하는 시간(분)을 입력하고 "확인"을 누르세요.

### Q. 점수를 잘못 입력했어요.

- 라이브 스코어보드: **- 버튼**으로 1점씩 차감합니다.
- 경기 카드: 점수 입력란을 직접 수정 후 "저장"합니다.

### Q. 대회를 실수로 종료했어요.

Admin 페이지에서 해당 대회 선택 후 **"대회 재개"** 버튼을 누르면 됩니다.

### Q. 삭제한 경기를 복구할 수 있나요?

삭제된 경기는 복구할 수 없습니다. 새로 "경기 추가"를 해주세요.

### Q. 여러 대회를 동시에 운영할 수 있나요?

네. Admin에서 대회를 여러 개 생성하고, 드롭다운으로 전환하며 관리할 수 있습니다.
단, 라이브 기록은 한 번에 하나의 경기만 가능합니다.
